UPLOAD ALL SEVEN FILES TO THE ROOT OF:
https://github.com/craigwhitson/SA55IMC-Craigs-WorldCup

1. On your laptop, open the repository.
2. Select Add file > Upload files.
3. Upload all files from this package.
4. Confirm replacement of the existing index.html and calendar.html files.
5. Commit directly to the main branch.
6. Wait 2-5 minutes.
7. Open the PUBLISHED SITE in Safari or Chrome:
   https://craigwhitson.github.io/SA55IMC-Craigs-WorldCup/

IMPORTANT:
Do not open index.html inside the GitHub mobile app. That is a file preview and may not run the site styling or scripts correctly.
