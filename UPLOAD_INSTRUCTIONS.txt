UPLOAD
1. Extract this ZIP on your laptop.
2. Open the GitHub repository above.
3. Select Add file > Upload files.
4. Upload every file from the extracted folder into the repository root.
5. Confirm replacement of existing files.
6. Commit directly to the main branch.
7. Wait about 2–5 minutes.
8. Open:
   https://craigwhitson.github.io/SA55IMC-Craigs-WorldCup/

IMPORTANT CACHE STEP
Because an earlier service worker may be stored on your phone:
- Open the website in Safari.
- Refresh it once.
- If the old version still appears, go to iPhone Settings > Safari > Advanced > Website Data,
  search for github.io, remove that site data, and reopen the address.
- Remove the old Home Screen icon and add it again after the new version is visible.

SHARING
